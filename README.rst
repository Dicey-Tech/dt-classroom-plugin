dt-classroom-plugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/Dicey-Tech/dt-classroom-plugin

Usage
-----

::

    tutor plugins enable dt-classroom-plugin
